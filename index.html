Elevator Controller: Python to Vivado HLS Conversion

The project details converting a Python elevator controller to Vivado High-Level Synthesis (HLS) for FPGA, highlighting differences between software (dynamic structures, unbounded loops) and hardware needs (fixed-size arrays, bounded control flow). Success required replacing Python's heap-based approach with a simplified FIFO array implementation, explicit loop bounds, and fixed-point data types for a functional HLS C++ design.
